========================================
AI Agent 聊天前端 - 快速启动指南
========================================

一、前端启动步骤：
------------------
1. 进入前端目录
   cd front

2. 安装依赖（如果尚未安装）
   npm install

3. 启动开发服务器
   npm run dev

4. 访问地址
   http://localhost:3000


二、后端启动：
--------------
请确保后端服务运行在 http://localhost:8080

后端接口地址：
- GET  /ai/queryAllSessions
- GET  /ai/queryHistoryMessages/{sessionId}
- POST /ai/createSession
- POST /ai/chat


三、功能说明：
--------------
1. 新建聊天：
   - 点击左侧"新建聊天"按钮
   - 填写表单（欢迎消息、系统提示词、模型配置等）
   - 提交后自动选中并打开该聊天

2. 发送消息：
   - 在底部输入框输入消息
   - 按 Enter 键或点击发送按钮
   - 支持流式响应显示

3. 历史聊天：
   - 左侧显示所有历史聊天列表
   - 点击可切换到对应会话
   - 支持按时间分组（今天、过去7天）

4. 样式主题：
   - 采用 ChatGPT 风格深色主题
   - 黑白配色方案


四、注意事项：
--------------
1. multiModal 选项建议保持关闭（当前未实现文件上传）
2. SSE 流式聊天功能需要后端支持
3. 如遇跨域问题，请检查 vite.config.js 代理配置


五、项目技术栈：
---------------
- Vue 3 (Composition API)
- Vite (构建工具)
- Axios (HTTP 客户端)
- CSS3 (ChatGPT 风格样式)


更多详细信息请查看 README.md
